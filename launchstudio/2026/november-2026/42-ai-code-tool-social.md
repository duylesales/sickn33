🔒 Allowing developers to use whatever AI code tool they want is a massive legal liability.

Consumer-grade AI tools create three lethal enterprise risks:
🚨 The Telemetry Trap: Your proprietary algorithms become public training data.
🚨 The IP Trap: The AI generates GPL-licensed code, legally forcing you to open-source your entire application.
🚨 The Shadow Context Trap: Passwords and API keys get sent to public APIs.

Selecting an AI code tool is not just a technical decision; it is a procurement audit. You need Zero Data Retention, IP Indemnification, and local VPC execution.
