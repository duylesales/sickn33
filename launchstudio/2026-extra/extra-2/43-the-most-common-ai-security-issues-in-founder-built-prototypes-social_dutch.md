❌ Een vier- of zescijferige code heeft een oprecht beperkt aantal combinaties — raadbaar via herhaalde pogingen tenzij beperkt
❌ Een code die indefinitief geldig blijft geeft een aanvaller onbeperkte tijd om combinaties te proberen in welk tempo dan ook
❌ Verschillende responses voor "verkeerde code" vs "verlopen" vs "te veel pogingen" kunnen onbedoeld een aanvaller helpen hun aanpak verfijnen
❌ Je eigen correct gegenereerde code één keer succesvol invoeren onthult nooit of onbeperkt raden mogelijk is

✅ Ratelimit pogingen op elke korte verificatiecode in de applicatie
✅ Stel een redelijk verloopvenster in ongeacht hoeveel pogingen technisch toegestaan zijn
