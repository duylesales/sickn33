✅ **The 20-point offshore partner evaluation checklist every CTO needs:**

Score each 0-3. Below 45/60 = risk. Below 35/60 = disaster.

**🔧 Technical (18 pts max)**
□ Tech stack alignment — production projects in YOUR stack
□ Architecture — can sketch a design during the call
□ Code quality — live coding or sample code review
□ DevOps — CI/CD, containers, IaC
□ Testing — minimum coverage standards, dedicated QA
□ Security — SAST/DAST, dependency scanning

**📋 Process (15 pts max)**
□ Agile workflow — planning, standups, reviews, retros
□ Documentation — ADRs, API docs, onboarding guides
□ Change management — formal scope change process
□ Estimation accuracy — historical data available
□ Incident response — SLAs and escalation paths

**👥 People (12 pts max)**
□ Team transparency — interview actual engineers
□ Retention — avg tenure >18 months
□ English proficiency — technical discussions fluent
□ Cultural bridge — PM who speaks your language

**📑 Legal (15 pts max)**
□ IP ownership — explicit transfer clause
□ Governing law — Dutch/UK/SG jurisdiction
□ GDPR DPA — compliant data processing
□ Exit clause — code handover process defined
□ Replacement SLA — <2 weeks

**How to use:** Evaluate 3-5 partners. Pilot the top 2 for 2-4 weeks. Then commit.
